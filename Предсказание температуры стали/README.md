Задача: Необходимо построить модель, которая предскажет температуру стали. <br>

Выполнено: Подготовка данных, поиск аномалий. Созданы признаки, первый и последний замер температуры, сумма мощности каждой партии для снижения вычислительной нагрузки.<br>
Выполнена оценка мультиколлинеарности. Проведено обучение модели LinearRegression, RandomForestRegressor, CatBoostRegressor, LGBMRegressor, XGBRegressor.<br>
